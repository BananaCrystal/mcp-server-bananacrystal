# Supported Currencies

All currencies are fully-backed stablecoins on Hedera. The token symbol is the ISO currency code with a `b` suffix (e.g. Nigerian Naira NGN → **NGNb**).

Use `list_available_tokens` via the MCP server to get the live list with current Hedera token IDs and exchange rates.
